掌握技术

  • 编程语言 : c语言、JAVA、python
  • 前端开发 : HTML5+CSS+Js(ES6)
  • 前端框架 : Vue全家桶(Vue-cli、Vuex、vue-router、Axios、Element)、微信小程序
  • 后端框架 : springBoot、Mybatis、swagger2、mybatis-plus
  • 数据库 : Mysql、redis
  • 开源项目 : RabbitMQ、Kafka、Zookeeper、hexo、picGo

本科项目

以下项目为本科学习、比赛的项目，只描述项目中自己承担的部分

毕设-英语线上学习系统

Vue3 Element-plus axios
  后端请用SpringBoot+MyBatis+Swagger2等框架、前端使用Vue3框架+Element-plus组件库+Axios网络请求 库、第三方服务使用七牛云存储以及短信云短信服务，数据库使用MySql以及使用Redis作为缓存使用，其中实现 了“在线浏览视频”、“上传视频”、“上传资料”、“评论信息”、“用户信息”等功能

手机网页版服装商店

Vue3 git 移动端
  2020年寒假，在自学Vue.js开发手机网页端的服装商店，个人主要负责主要前端代码的实现，包括首页界面、商品详情页、加入购物车等界面与功能的实现，并将代码提交到GitHub上面。

校园代拿系统

微信小程序 云开发 移动端
  在参加互联网+大赛，使用微信小程序云服务器开发校园代拿系统，个人主要负责“数据库的分析与设计”、“用户登录功能”、“订单管理”等功能以及界面的实现。

课设-简易版网易云MV音乐

Vue3 移动端
  2021年，使用Vue.js开发简易版网易云MV音乐（后端接口调用的是网易云提供的接口），其中实现了首页MV推荐、详情页、评论以及个人界面等基础功能。

校企合作-简易版微博

pc端 JSP
  2021年，在学校举行的校企合作中，使用SpringBoot+Mybatis+JSP设计”简易版微博“的PC端网页，主要负责前端界面的的编写以及后端部分接口的实现。

我的校园平台

springBoot 微信小程序
  使用SpringBoot+Mybatis作为后台设计“我的校园平台”微信小程序，主要负责内容展示、发表评论、点赞等功能功能接口，并部分前端代码。

图片管理系统

springBoot JPA mysql
  使用HTML+SpringBoot（使用JPA作为持久层框架）开发图片管理系统，主要负责数据库的分析与设计与后端接口的实现。

第一个课设-书店管理系统

JAVA JDBC
  使用JAVA与JDBC开发书店管理系统，采用原生的Java视图编写界面，主要负责“数据库的分析与设计”、“入库管理功能”、“书籍管理功能”、“销售管理功能”等功能。
