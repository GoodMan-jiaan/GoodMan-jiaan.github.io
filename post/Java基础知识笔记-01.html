名词解释

JVM

java虚拟机：加载.class字节码文件并运行.class字节码文件

JRE

Java运行环境；包含JVM和运行Java所必需的Java系统类库

JDK

Java开发工具包，包含JRE和开发java程序所必须的命令工具

说明

  • 运行Java程序的最小环境为JRE
  • 开发Java程序的最小环境为JDK

命名规则

  • 只能包含字母、数字、_和$符，并且不能以数字开头

  • 严格区分大小写

  • 不能使用关键字

  • 允许中文命名，但不建议，

  • 建议”英文的见名知意”、”驼峰命名法”

数组

数组是数据的集合，一个容器，用来存储任何类型的数据，包括原始数据类型和引用数据类型，但是一旦指定了数组的类型之后，就只能用来存储指定类型的数据。

数组声明的三种方式

  • 数据类型[] 数组名 = new 数据类型[长度];
  • 数据类型[] 数组名 = {数据,数据,…,数据};
  • 数据类型[] 数组名 = new 数据类型长度[] {数据,数据,…,数据};

一维数组

  • 数组变量的声明：

    语法：数据类型[] 数组名；如 int[] num;

  • 数组对象的创建：

    语法：数组名 = new 数据类型[长度]; 如：num = new int[4];数组声明其长度后不可改变

  • 赋值：

    语法：数组名[下标] = 数据;如：num[0] = 3;

二维数组

  • 数组变量的声明：
    语法：数据类型[][] 数组名;，如：int[][] num;

  • 数组对象的创建：
    语法：数组名 = new 数据类型[外长度][内长度];，如：num = new int[4][3];，数组声明其长度后不可改变

  • 赋值：
    语法：数组名[外下标][内下标] = 数据;，如：num[0][0]= 3;
