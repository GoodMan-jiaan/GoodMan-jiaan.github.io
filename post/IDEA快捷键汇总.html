IDEA快捷键汇总

发表于2020-10-03|更新于2024-09-17|开发基础
|字数总计:178|阅读时长:1分钟|阅读量:

IDEA常用快捷键

  1. psvm + Tab键 / main + Tab键 生成主函数public static void main(String[] args)

  2. sout+Tab键 生成System.out.println()语句

  3. Ctrl + X 删除当前行

  4. Ctrl + D 复制当前行

  5. Alt + Insert 生成get,set方法，构造函数等

  6. Alt + 回车 导入包，自动修正

  7. Ctrl + Alt + L 格式化代码

  8. Alt + <-左右->键 实现窗口左右转换

  9. Ctrl + 鼠标点击 快速找到成员变量的出处

查找

  1. Ctrl + F 在当前文件进行文本查找
  2. Ctrl + R 在当前文件进行文本替换
  3. Shift + Ctrl + F 在项目进行文本查找
  4. Shift + Ctrl + R 在项目进行文本替换
  5. Shift + Shift 快速搜索
  6. Ctrl + N 查找class
  7. Ctrl + Shift + N 查找文件
