Linux学习笔记

权限管理命令

chmod命令

  • 功能：改变目录或文件权限

  • 语法：chmod [{ugoa}{+-=}{rwx}] [文件或目录]

    chmod [421] [文件或目录]（八进制）

  • 选项：-R 递归修改

符号模式

who 用户类型 说明
u user 文件所有者
g group 文件所有者所在组
o others 所有其他用户
a all 所有用户, 相当于 ugo

operator的符号模式

Operator 说明
+ 为指定的用户类型增加权限
- 去除指定用户类型的权限
= 设置指定用户权限的设置，即将用户类型的所有权限重新设置

permission的符号模式

模式 名字 说明
r 读 设置为可读权限
w 写 设置为可写权限
x 执行权限 设置为可执行权限
X 特殊执行权限 只有当文件为目录文件，或者其他类型的用户有可执行权限时，才将文件权限设置可执行
s setuid/gid 当文件被执行时，根据who参数指定的用户类型设置文件的setuid或者setgid权限
t 粘贴位 设置粘贴位，只有超级用户可以设置该位，只有文件所有者u可以使用该位

八进制语法

chmod命令可以使用八进制数来指定权限。文件或目录的权限位是由9个权限位来控制，每三位为一组，它们分别是文件所有者（User）的读、写、执行，用户组（Group）的读、写、执行以及其它用户（Other）的读、写、执行。历史上，文件权限被放在一个比特掩码中，掩码中指定的比特位设为1，用来说明一个类具有相应的优先级。

# 权限 rwx 二进制
7 读 + 写 + 执行 rwx 111
6 读 + 写 rw- 110
5 读 + 执行 r-x 101
4 只读 r– 100
3 写 + 执行 -wx 011
2 只写 -w- 010
1 只执行 –x 001
0 无 — 000

例如， 765 将这样解释：

  • 所有者的权限用数字表达：属主的那三个权限位的数字加起来的总和。如 rwx ，也就是 4+2+1 ，应该是 7。
  • 用户组的权限用数字表达：属组的那个权限位数字的相加的总和。如 rw- ，也就是 4+2+0 ，应该是 6。
  • 其它用户的权限数字表达：其它用户权限位的数字相加的总和。如 r-x ，也就是 4+0+1 ，应该是 5。

chown命令

  • 作用：改变文件或目录的所有者
  • 语法：chown [用户] [文件或目录]

chgrp命令

  • 作用：改变文件或目录的所属组
  • 语法：chgrp [用户组] [文件或目录]

umask命令

  • 作用：显示，设置文件的缺省权限
  • 语法：umask
  • 选项：-S 以rwx形式显示新建文件的缺省权限

文件处理命令

命令格式

  • 语法：命令 [-选项] [参数] 例：ls   -la   /etc
  • 说明：
    1. 个别命令使用不遵循此格式
    2. 当有多个选项时，可以写在一起
    3. 简化选项与完整选项 -等于 –all

ls

  • ls：list
  • 命令所在路径：/bin/list
  • 功能：显示目录文件
  • 主要选项：-a：显示所有文件 -l：显示详细信息 -d：查看目录属性
  • 参数：指定要显示的目录

目录处理命令

mkdir

  • 语法：mkdir -p [目录名]
  • 功能描述：创建新目录；选项-p为递归创建，用于多重新目录的创建

cd

  • 语法：cd[目录]
  • 功能描述：切换至对应目录

pwd

  • 功能：显示当前目录

rmdir

  • 语法：rmdir [目录名]
  • 功能：删除==空==目录，很少用

cp

  • 语法：cp [原文件或目录] [目标目录]

  • 选项：-r：复制目录； -p：保留文件属性

  • 功能：复制文件或目录

mv

  • 语法：mv [原文件或目录] [目标目录]
  • 功能：剪切文件，改名
  • 补充：ctrl+L或clear命令可以清屏

rm

  • 语法：rm [目录或文件]
  • 选项：-r删除目录 -f强制执行
  • 功能：本意是删除文件，但也可以删除目录，要加上选项-r

文件处理命令

touch

  • 语法：touch [文件名]
  • 功能：创建空文件

cat

  • 语法：cat [文件名]

  • 选项：-n显示行号

链接命令

ln 生成链接文件

  • 语法：ln -s [原文件] [目标文件]；不加-s选项则会创建硬链接
  • 选项：-s 创建软链接

压缩解压命令

tar

  • 语法：tar [选项] [-f 压缩包名] 源文件或目录
  • 功能：”.tar”格式的打包和解打包都使用 tar 命令，区别只是选项不同
  • 选项：
    1. -c：打包；
    2. -f：指定压缩包的文件名。压缩包的扩展名是用来给管理员识别格式的，所以一定要正确指定扩展名；
    3. -v：显示打包文件过程；
